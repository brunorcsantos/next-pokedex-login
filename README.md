# 📘 Next Pokedex Login

A modern **Pokédex web application** built with **Next.js**, featuring a simple authentication flow before accessing the application.

🔗 **Live Demo:** https://next-pokedex-login.vercel.app/  
🔗 **Repository:** https://github.com/brunorcsantos/next-pokedex-login

---

# 🚀 About the Project

**Next Pokedex Login** is a web application that allows users to authenticate and then access a Pokédex interface displaying Pokémon data.

The project was created to practice **full-stack development concepts**, including authentication, API consumption, and modern frontend architecture using **Next.js**.

---

# ✨ Features

- 🔐 User authentication (login screen)
- 📱 Responsive interface
- 🧩 Pokémon listing
- 🔎 Pokémon information visualization
- ⚡ Modern React / Next.js architecture
- ☁️ Deployment using Vercel

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/brunorcsantos/next-pokedex-login.git

Navigate to the project folder:
cd next-pokedex-login

Install dependencies:
npm install

Run the development server:
npm run dev

Open your browser at:
http://localhost:3000

🌐 Deployment

The application is deployed using Vercel, which provides optimized hosting for Next.js applications and automated deployments from GitHub repositories.

Live application:
